Java Comments

Single-line latest Single-line comments start with two forward slashes (//). Any text between // and the end of the line is ignored by Java (will not be executed).

Java Multi-line Comments Multi-line comments start with /* and ends with */. Any text between /* and */ will be ignored by Java.
